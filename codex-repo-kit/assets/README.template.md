# Everything Codex Agentic Coding

A curated, production-ready collection of agentic coding assets (agents, skills, commands, rules, hooks, and configs) designed for Codex-style workflows across multiple languages.

## What's Included

- **Agents**: Role-specific agent definitions for backend, frontend, QA, and architecture workflows.
- **Skills**: Reusable playbooks for analysis, refactoring, debugging, testing, and release tasks.
- **Commands**: High-signal commands for planning, execution, and verification loops.
- **Rules**: Common and language-specific conventions for reliable, secure delivery.
- **Hooks & Contexts**: Memory persistence and project onboarding helpers.
- **MCP Configs**: Example tool and data source integrations.

## Quick Start

```bash
# Clone your new repo
git clone <your-repo-url>

# Browse skills, agents, and commands
ls agents skills commands
```
