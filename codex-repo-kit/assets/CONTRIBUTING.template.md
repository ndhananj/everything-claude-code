# Contributing

Thanks for contributing to this agentic coding repository!

## Guidelines

- Keep additions consistent with the existing directory structure.
- Prefer reusable, framework-agnostic patterns.
- Include verification steps for skills and commands when possible.
- Document any external dependencies in the relevant README or docs.

## Submitting Changes

1. Create a feature branch.
2. Make changes with clear commits.
3. Open a pull request describing the intent and testing notes.
